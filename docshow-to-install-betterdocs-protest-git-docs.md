<h2 class="wp-block-heading" id="0-toc-title"><strong>How To Install BetterDocs PRO</strong></h2>
<!-- /wp:heading -->

<!-- wp:paragraph -->
<p><strong>Step 1:</strong> Make sure that you have <a href="http://betterdocs.co/docs/activate-betterdocs/" target="_blank" rel="noreferrer noopener"><strong>Installed and Activated the Free version of BetterDocs</strong></a>. You will need to keep the Free version activated to use the PRO Version.</p>
<!-- /wp:paragraph -->

<!-- wp:image {"id":13523} -->
<figure class="wp-block-image"><img src="https://betterdocs.co/wp-content/uploads/2019/10/1-Doc-Update-How-to-Install-BetterDocs-PRO-BetterDocs.png" alt="How To Install BetterDocs PRO step 1" class="wp-image-13523" title="How to Install BetterDocs PRO? 1"/></figure>
<!-- /wp:image -->

<!-- wp:paragraph -->
<p><strong>Step 2:</strong> Go to your <a href="https://wpdeveloper.com/account/" target="_blank" rel="noreferrer noopener"><strong>WPDeveloper account</strong></a> and under the <strong>“Downloads” </strong>tab, you will see your downloadable files. Afterward, download the plugin file. It should look like <strong>betterdocs-pro.zip.</strong></p>
<!-- /wp:paragraph -->

<!-- wp:image {"id":13524} -->
<figure class="wp-block-image"><img src="https://betterdocs.co/wp-content/uploads/2019/10/2-Doc-Update-How-to-Install-BetterDocs-PRO-BetterDocs.png" alt="How To Install BetterDocs PRO step 2" class="wp-image-13524" title="How to Install BetterDocs PRO? 2"/></figure>
<!-- /wp:image -->

<!-- wp:paragraph -->
<p><strong>Step 3:</strong> Afterward, Go to your <strong>Plugins</strong> page from <strong>WordPress Dashboard. </strong>Then click on the <strong>‘Add New’</strong> option and Upload the BetterDocs PRO plugin that you have downloaded. After successfully uploading, click on the <strong>‘Install Now’</strong> button.</p>
<!-- /wp:paragraph -->

<!-- wp:image {"id":13525} -->
<figure class="wp-block-image"><img src="https://betterdocs.co/wp-content/uploads/2019/10/3-Doc-Update-How-to-Install-BetterDocs-PRO-BetterDocs.png" alt="How To Install BetterDocs PRO step 3" class="wp-image-13525" title="How to Install BetterDocs PRO? 3"/></figure>
<!-- /wp:image -->

<!-- wp:paragraph -->
<p><strong>Step 4:</strong> After installation is finished, simply click on the <strong>‘Activate’</strong> button.</p>
<!-- /wp:paragraph -->

<!-- wp:image {"id":13526} -->
<figure class="wp-block-image"><img src="https://betterdocs.co/wp-content/uploads/2019/10/4-Doc-Update-How-to-Install-BetterDocs-PRO-BetterDocs.png" alt="How To Install BetterDocs PRO step 4" class="wp-image-13526" title="How to Install BetterDocs PRO? 4"/></figure>
<!-- /wp:image -->

<!-- wp:paragraph -->
<p>After activation, navigate to the plugin to use all the PRO features of BetterDocs. To configure the basic settings, go to the ‘<strong>Settings</strong>’ tab under the app. If you want to configure analytics,&nbsp; go to the ‘<strong>Analytics</strong>’ tab.&nbsp;&nbsp;</p>
<!-- /wp:paragraph -->

<!-- wp:image {"id":13527} -->
<figure class="wp-block-image"><img src="https://betterdocs.co/wp-content/uploads/2019/10/5-Doc-Update-How-to-Install-BetterDocs-PRO-BetterDocs.gif" alt="Go to BetterDocs settings" class="wp-image-13527" title="How to Install BetterDocs PRO? 5"/></figure>
<!-- /wp:image -->
